Программа, которая решает квадратные уравнения.

Пользователю предлагается стандартный шаблон: ax² + bx + c = 0.
Пользователю предлагается ввести значения a, b и c.
Если введённые пользователем значения корректны, выдаёт ответ.

Обратите внимание, что некоторые значения, хоть и являются корректными, противоречат определению квадратного уравнения!
Например, при a=0 уравнение уже не будет квадратным!

Программа решает через дискриминант.
Умеет работать и при D>0, и при D=0, и при D<0.
