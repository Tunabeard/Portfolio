Обыкновенные звёздочки для выставления рейтинга.
Пользователь наводит курсор на звезду, после чего закрашивается эта звезда и все идущие перед ней.

Сложность в том, что во внутреннем инструментарии CSS не существует способа применить стилевые свойства для элементов, которые идут ПЕРЕД :hover-элементом внутри контейнера.
Стилевые свойства можно изменять только для тех элементов, которые идут ПОСЛЕ :hover-элемента внутри контейнера.
Здесь это очень мешает. Приходится хитрить.

Предлагаю два способа (файлы index1 и index2).
