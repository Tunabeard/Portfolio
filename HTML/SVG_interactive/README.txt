Интерактивная SVG-картинка.
При поднесении курсора к некоторым элементам меняются их стилевые свойства.

Полный список интерактивных элементов:
1) Труба — убирается дым;
2) Крыша — крыша переворачивается, появляется надпись;
3) Окно — окно открывается, за ним сидит чудище;
4) Облака — начинается дождь;
5) Солнце — облака убираются, появляются лучи;
6) Яблоки — выбранное яблоко падает на траву.

Все элементы рисовались без использования сайтов-генераторов — только ручным вводом кода.
